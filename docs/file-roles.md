# File Roles

## `.tasks/issues.json`
- **Role:** Raw cached GitHub API response.
- **Written by:** `bin/board-pull` only.
- **Read by:** `bin/board-render`.
- **Never hand-edited.** Re-running `board-pull` overwrites it cleanly.

## `.tasks/board.json`
- **Role:** Single local source of truth (derived from issues).
- **Written by:** `bin/board-render`.
- **Read by:** skills (`board-plan`, `board-run-ready`), hooks.
- **Schema:** `[{ "number": int, "title": str, "status": str, "labels": [str], "url": str, "assignees": [str] }]`
- **Status values:** One of the canonical enum (`inbox`, `ready`, `in-progress`, `needs-review`, `blocked`, `needs-info`, `done`).
- **Never hand-edited.**

## `TODO.md`
- **Role:** Read-only human-readable render of the board.
- **Generated by:** `bin/board-render`.
- **Header:** `<!-- generated by board-render -- do not edit -->`
- **Structure:** Tasks grouped by status in canonical order, sorted by issue number within each group.
- **Never hand-edited.** Any manual changes will be overwritten on next render.
